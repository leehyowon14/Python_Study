##### 새로 알게된 것.

방향마다 케이스를 나누는 것은 매우 비효율적이며, 코드를 짜는 데 불편이 있음.
방향을 단위벡터로 변환하여 사용하는 것이 좋아보임.

### 첫번째 풀이

SCORE: 0.0 / 100

> **SELF-FEEDBACK**
> 케이스를 나누지 않고 하는 방법을 강구해보아야 할 듯
> -> 방향을 단위 벡터 형식으로 변환하여 처리
